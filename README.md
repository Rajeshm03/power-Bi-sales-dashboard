# Tailwind Traders Power BI Dashboard

This Power BI report was created as part of the **Microsoft Power BI Data Analyst Capstone Project** on Coursera.

## 📊 Project Overview

The dashboard analyzes key performance metrics for **Tailwind Traders**, a fictional retail company. It includes:

- Total Revenue, Net Profit, and Profit Margin KPIs
- Sales Trends over time
- Category-wise performance
- Interactive filtering using date slicers

## 🛠️ Tools Used

- Power BI Desktop
- DAX
- Power Query
- Calendar Table (for time intelligence)
